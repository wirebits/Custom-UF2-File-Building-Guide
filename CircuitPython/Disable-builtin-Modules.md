# Disable-builtin-Modules

# It disables builtin modules while building CircuitPython `uf2` firmware.

# Steps
1. Go to `ports/board_name_folder` folder.
2. Then, go to `boards` folder.
3. Choose your board folder.
4. Go inside the folder.
5. Open `mpconfigboard.mk` file.
6. Just add `builtin-Modules` by just copy given below list in `mpconfigboard.mk` file.
7. Save this file.
8. Done! Build, flash and it disable most of builtin-Modules in the firmware now!
- It reduce the size of firmware file.
